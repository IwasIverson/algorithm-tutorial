# algorithm-tutorial

> 算法、数据结构这类知识点真的需要天分。本人不是算法工程师，也做不了算法工程师。总是在见识到一些算法大神的优雅实现后，感叹于自己是多么的弱智，同样是写代码，我写的是什么垃圾！
>
> 这个项目的初衷，是将工作中经常会接触到的一些数据结构和算法整理归纳。人总是要有理想的，要想少写垃圾的代码，必须夯实算法的基础。路漫漫兮其修远，吾将上下而求索。
>
